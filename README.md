# Niffler

> Niffler is Chasing the Shiny Meme Coins.

### Development

```bash
# Required: typescript watch compilation
$ pnpm watch

# Required: development server with hot reload (nodemon)
$ pnpm dev

# Format with prettier
$ pnpm format
```

### Production

```bash
# build for production
$ pnpm build

# start production app
$ pnpm start
```
